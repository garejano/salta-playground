components: Componentes basicos, podem ser usados via template

dynamic-componets: Componentes que nao sao usados em template, servem para ser 'injetaveis' dinamicamente via TS
  exemplos de uso: form.component.ts e no filtro-module/filtro.component.ts

filtro: modulo com o filtro padrao em 2025. utiliza os componentes de dynamic-components

formulario: modulo com o gerador de formualrios no padrao do DS. utiliza os componentes de dynamic-components
